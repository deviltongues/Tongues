What is DevilTongues?

DevilTongues is a sophisticated financial analytics platform that identifies arbitrage opportunities in synthetic bonds through real-time put-call parity analysis. By connecting to LSEG Refinitiv's enterprise-grade market data, the platform calculates implied risk-free rates from options pricing and flags profitable conversion and reverse conversion strategies.

What are Synthetic Bonds?
Synthetic bonds are created by combining options positions to replicate the payoff of holding (or shorting) the underlying stock. When these synthetic positions are mispriced relative to the actual stock, arbitrage opportunities emerge that our platform automatically detects.

How It Works

1

Fetch Real-Time Market Data

Connect to LSEG Refinitiv to retrieve live spot prices and comprehensive options chains for major equity underlyings (AAPL, MSFT, TSLA, NVDA, etc.).

2

Calculate Implied Risk-Free Rates

Use put-call parity to compute implied rates from options pricing: C - P = S - K√óe^(-rT). Compare these against benchmark risk-free rates.

3

Identify Arbitrage Opportunities

Detect mispricings when implied rates deviate from benchmarks. Flag profitable conversion (buy synthetic + short stock) and reverse conversion (sell synthetic + buy stock) strategies.

4

Analyze & Execute

Review detailed strategy breakdowns with position requirements, profit expectations, risk assessments, and execution cost calculations to make informed trading decisions.

Getting Started

Prerequisites

  • Python 3.12 (3.13 not yet supported by all dependencies)
  • LSEG Refinitiv Data API credentials
  • LSEG Workspace or Eikon desktop application running and logged in

üìç Important: The application will start on http://127.0.0.1:8000 by default. Ensure LSEG Workspace or Eikon is running and you're logged in before launching.

First-Time Usage

  1. Market Data Tab: Select an underlying (e.g., MSFT.O) and click "FETCH SPOT PRICE" to retrieve current stock price
  2. Configure Options Chain: Set Min/Max Strike prices (e.g., 400-500), Min/Max Expiry dates, and click "SCAN OPTIONS CHAIN"
  3. Analysis Tab: Set Risk-Free Rate (e.g., 5.0%) and Arbitrage Threshold (e.g., 0.5%), then click "ANALYZE ARBITRAGE"
  4. Review Results: Click on detected opportunities to see detailed strategy breakdowns and use the Execution Calculator to model costs

Strategy Explanation

The platform exploits deviations from put-call parity, a fundamental no-arbitrage relationship:

C - P = S - K √ó e^(-rT)

Where C = Call price, P = Put price, S = Spot price, K = Strike price, r = Risk-free rate, T = Time to expiration.
